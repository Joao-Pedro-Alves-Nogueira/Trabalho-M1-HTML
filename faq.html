2. Dúvidas sobre Cadastros

Como cadastrar um cliente sem e-mail?

O campo de e-mail é obrigatório para o envio de notas fiscais. Caso o cliente não possua, utilize o padrão nao@possui.com.br para prosseguir com o registro.

Errei o ISBN no cadastro de um livro, o que fazer?

Nesse caso, você deve notificar a Gerência de Estoque. Atualmente, por segurança, apenas usuários administrativos podem editar o código ISBN após o salvamento inicial.

3. Operações de Venda

Quais formas de pagamento o sistema aceita?

O sistema está configurado para aceitar Pix, Cartão de Crédito, Débito e Dinheiro. Certifique-se de selecionar a opção correta na página de Registro de Venda para não dar erro no fechamento de caixa.

Como realizar a troca de um livro?

Consulte a nossa página de Políticas de Troca. Lá você encontrará o passo a passo ordenado para realizar o procedimento sem invalidar o estoque.

Ainda precisa de ajuda?

Se o problema persistir ou o sistema apresentar lentidão, entre em contato com o TI pelo ramal #404 ou envie um e-mail para suporte@paginacerta.com.br.
